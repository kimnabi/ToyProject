# 제품 판매 미니 웹 앱 개발
이 레포지토리는 인프런의 개발자의품격님의 VUEjS 을 보며 실습하고 정리하여 
, 상품 등록이나 장바구니 구현, 이미지 및 데이터 조회 등 핵심이 되는 기능을 모두 구현하게 연습했습니다
