# 제품 판매 미니 웹 앱 개발

제품 판매를 위한 전체 기능을 구현하는 것은 아니지만, 상품 등록이나 장바구니 구현, 이미지 및 데이터 조회 등 핵심이 되는 기능을 모두 구현하게 됩니다.

제품 판매 웹을 만드는 이유는 일반적인 웹 애플리케이션을 만드는 데 필요한 기능 대부분을 포함하고 있기 때문입니다. 거의 대부분의 웹 애플리케이션이 다음 기능을 포함합니다
